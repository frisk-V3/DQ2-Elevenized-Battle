#  DQ2 Elevenized Battle Engine

DQ2の世界観に、**DQ11のシステム**を強引にねじ込んだ改造バトルエンジン。

---

##  バトルシステム (Features)

- **ゾーン (Zone)**
  - 行動や被弾でゲージが蓄積。
  - 発動するとステータスが1.5倍になり、名前が青白く発光。
- **れんけい技 (Co-op Skills)**
  - パーティ全員がゾーンに入ると解放。
  - **「ゴールデンブレイク」**などの超強力な一撃を放つ。
- **高防御ボス (Golden Slime)**
  - HP: 30 / 守備力: 500 / 回避率: 30%
  - 通常攻撃は1しか通らないが、呪文（デイン）なら貫通可能。

---

##  操作コマンド (Commands)

1. **こうげき** (`Attack`)
   - 守備力の影響を受ける物理攻撃。
2. **デイン** (`Spell`)
   - MPを8消費。守備力を無視して10ダメージ。
3. **れんけい** (`Renkei`)
   - ゾーン中のみ出現。防御を貫通して20以上のダメージ。
4. **ぼうぎょ** (`Defend`)
   - 受けるダメージを軽減し、生存率を高める。

---

##  技術仕様 (Technical)

- **描画**: SVG (GitHubリポジトリからRawデータをFetch)
- **演出**: CSS3 Animation (シェイク・フェード・発光)
- **言語**: Vanilla JavaScript (外部ライブラリなし)

---

## 導入方法

1. `index.html` を作成する。
2. [こちらのリポジトリ](https://github.com)のコードを貼り付ける。
3. ブラウザで開くだけで、即バトル開始！
